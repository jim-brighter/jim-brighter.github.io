BRICK BLASTER 6000

An iOS game made with Objective-C and Cocos2D. The player controls a spaceship and defends earth from an asteroid bombardment. The game ends when earth gets hit by an asteroid; otherwise, keep playing to get your score as high as possible. Unpublished due to unoriginal art and expensive developer fees.

DUCKDUCKGO EMACS CHEATSHEET

An Emacs commands cheatsheet Instant Answer for DuckDuckGo. Mostly HTML/CSS styling with some Perl scripting. Recently restyled to fit DuckDuckGo's cheatsheet template.

PENN OS

A Unix-like virtual operating system that runs on a Linux host. Complete with processes/threads, scheduling, a FAT filesystem, and a shell/command line. Created in C with three teammates.

THE MAGICAL ADVENTURES OF LEMONS

A text-based adventure game with multiple level areas, NPC's and enemies, a trading system, side quests, items, and stats. Also includes a combat system with skill and chance components and rapidly-increasing difficulty. Created in C++ with one partner.

BRICK BLASTER 5000

A Breakout clone with heavy object-oriented programming and a graphical interface. The first game I created, running on any desktop environment. Created with Java Swing.
